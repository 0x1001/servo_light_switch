# Servo light switch
Servo light switch

Ventilation control
--------------------------------------
Project is based on Digispark Attiny85 small board and SG 90 servo motor.
Every 20 min ventilation is turned on for 1 minute. 
Case and servo motor attachment is 3D printed.
